# TeamFinder — платформа для поиска команды над pet-проектами

## Описание проекта

TeamFinder — это веб-приложение, созданное для того, чтобы разработчики, дизайнеры, аналитики и другие IT-специалисты могли находить единомышленников для совместной работы над pet-проектами. Здесь можно не только публиковать свои идеи, но и присоединяться к уже существующим проектам, находить участников в команду, обсуждать задачи и развивать навыки в реальной командной работе.

### Основные возможности:

- **Создание проектов** — пользователи могут опубликовать свой проект, указав название, описание, необходимые навыки и статус (открыт/закрыт).
- **Поиск и фильтрация** — удобный фильтр по навыкам помогает быстро найти интересные проекты.
- **Участие в проектах** — можно откликнуться на проект и стать его участником.
- **Избранное** — сохраняйте понравившиеся проекты в избранное.
- **Личный кабинет** — редактируйте профиль, добавляйте аватар, указывайте контакты и информацию о себе.
- **Админ-панель** — полное управление пользователями и проектами.

## Используемые технологии

- Python 3.10
- Django 5.2
- PostgreSQL (или SQLite для локальной разработки)
- Docker, Docker Compose
- HTML, CSS 

## Быстрый запуск (с готовыми тестовыми данными)

Этот способ подходит, если вы хотите быстро ознакомиться с проектом, не вникая в настройки. Все тестовые данные уже подготовлены в дампе базы данных.

### 1. Распакуйте архив с проектом

### 2. Запустите контейнеры

```bash
docker compose up --build -d
```

**Восстановите базу данных с тестовыми данными**
```docker exec -i teamfinder_db psql -U teamfinder_user teamfinder_db < db_dump.sql```

**Откройте сайт в браузере**
http://localhost:8000

**Вход в админ-панель**
Адрес: http://localhost:8000/admin
Email: admin@example.com
Пароль: admin123

**Ручной запуск (без Docker)**

1. Создайте и активируйте виртуальное окружение
```python -m venv venv```

Для Windows:
```venv\Scripts\activate ```

Для Mac/Linux:
```source venv/bin/activate```

2. Установите зависимости:
```pip install -r requirements.txt```

3. Создайте файл .env:

POSTGRES_DB=teamfinder_db
POSTGRES_USER=teamfinder_user
POSTGRES_PASSWORD=teamfinder_password
POSTGRES_HOST=localhost
POSTGRES_PORT=5436
DJANGO_SECRET_KEY=ваш-секретный-ключ
DJANGO_DEBUG=True


4. Запустите PostgreSQL через Docker:
```docker compose up -d```

5. Выполните миграции:
```python manage.py migrate```

6. Создайте суперпользователя:
```python manage.py createsuperuser```

7. Запустите сервер:
```python manage.py runserver```

Сайт доступен по адресу: http://localhost:8000
